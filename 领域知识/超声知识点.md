# 超声知识点

## 1 超声模式有哪几种，分别有什么区别

超声设备通过不同的**工作模式**来提取人体组织的不同物理信息。临床上最核心的模式主要有四种：**B模式、M模式、多普勒模式（彩色、频谱）**，以及在此基础上发展的**弹性成像**等高级模式。

它们的主要区别在于**成像维度、显示方式**以及**揭示的生理特征**不同。

### 1. B模式（Brightness Mode，亮度模式）

这是所有超声检查的基础，即我们通常所说的“二维超声”或“黑白超”。

- **原理**：通过声束旋转扫查，接收回波的**强度**并根据强度赋予不同的**亮度**（强回声白，无回声黑），同时根据回波**时间**定位深度。
- **显示**：生成一幅**二维解剖结构切面图**（扇形、凸阵或线阵图像）。
- **用途**：显示器官的**形态、大小、边界、内部结构**以及有无占位性病变（如肿瘤、结石、积液）。
- **特点**：**实时动态**。医生可以一边移动探头，一边观察脏器的实时活动（如呼吸运动、探头的压迫变形）。

### 2. M模式（Motion Mode，运动模式）

这是超声中**时间分辨率最高**的模式，主要用来观察**高速运动的组织结构**。

- **原理**：它并不试图构建一幅完整的二维图，而是**仅选取B模式图像中的一条声束线**，将这条线上所有组织随时间推移产生的**运动轨迹**描绘出来。
- **显示**：横坐标代表**时间**，纵坐标代表**深度**。原本运动的组织（如心脏瓣膜）在M模式下显示为**波浪形或曲线**。
- **用途**：主要用于**心脏超声**。
  - **测量**：精确测量心室壁厚度、心腔大小。
  - **评估功能**：计算心脏的射血分数（EF值）、观察瓣膜（如二尖瓣）的开放与关闭速度是否正常。
- **区别**：相比于B模式（像在看电影），M模式更像是**“心电图式的解剖图”**，它能捕捉到毫秒级的运动变化，是心脏功能定量的金标准之一。

### 3. 多普勒模式（Doppler Mode）

此模式依据的是**多普勒效应**，主要用来评估**血流**或组织运动的速度与方向。

#### A. 彩色多普勒（Color Doppler，CDFI）

- **原理**：在B模式图像的基础上，将血流信息以**彩色编码**叠加显示。
- **显示**：通常设定**红色**代表血流方向朝向探头，**蓝色**代表血流方向远离探头。色彩越鲜艳（明亮），血流速度越快。
- **用途**：**半定量**观察。快速判断血管内有无血流、血流是否通畅、有无狭窄引起的“五彩镶嵌”状高速射流，或者有无反流（如瓣膜关闭不全）。

#### B. 频谱多普勒（Spectral Doppler）

它又分为**脉冲波多普勒（PW）** 和**连续波多普勒（CW）**。

- **原理**：在B模式或彩色模式下选定一个具体的取样点，以**波形图**的形式显示该点血流速度随时间的变化。
- **显示**：横坐标时间，纵坐标速度。基线以上为正（流向探头），以下为负（远离探头）。
- **区别与用途**：
  - **PW（脉冲波）**：能够**定位**深度（距离选通），但测量的最大速度有限。用于测量正常血管或低速血流（如静脉、门静脉）。
  - **CW（连续波）**：**不能定位**深度（沿声束线所有深度的叠加），但能测量**极高的速度**。用于测量高速血流（如心脏瓣膜狭窄时的射流、动脉狭窄处的峰值速度）。

#### C. 能量多普勒（Power Doppler，CDE/PDA）

- **原理**：显示血流中红细胞的**能量**（即反射信号的强度），不显示方向和速度。
- **显示**：通常呈现为单一色调（如橘黄或红色），没有色彩混叠。
- **用途**：对**低速、微弱血流**非常敏感。常用于检查实质器官（如肾脏、肝脏肿瘤）的微血管灌注情况，但对血流方向不敏感，且容易因组织运动产生伪像。

### 4. 高级/新兴模式

近年来随着技术进步，以下几种模式也越来越普及：

- **组织多普勒成像（TDI，Tissue Doppler Imaging）**：
  - 传统多普勒是看血流（快、低能量），组织多普勒是看**心肌壁**（慢、高能量）的运动速度。主要用于评估心脏舒张功能、心肌同步性。
- **弹性成像（Elastography）**：
  - **原理**：通过比较组织在受到轻微外力或声辐射力前后的**形变程度**（应变率），来反映组织的**软硬度**。
  - **用途**：主要用于肿瘤良恶性鉴别（恶性肿瘤通常比良性组织或正常组织更硬）。通常以彩色编码图（如红色软、蓝色硬）叠加在B模式图上显示。
- **超声造影（CEUS，Contrast-Enhanced Ultrasound）**：
  - 通过静脉注射微泡造影剂，利用**谐波成像**技术观察组织的**微循环灌注**。它可以实时显示肿瘤在不同时相（动脉期、门脉期、延迟期）的血流特征，用于肝癌等病变的诊断。

### 总结对比表

| 模式           | 核心依据          | 主要显示内容               | 临床优势                           |
| :------------- | :---------------- | :------------------------- | :--------------------------------- |
| **B模式**      | 回波强度、时间    | 二维解剖结构（黑白灰阶）   | 基础扫查，定位，形态学诊断         |
| **M模式**      | 运动轨迹          | 时间-深度曲线（波形）      | 高时间分辨率，精确测量心脏运动参数 |
| **彩色多普勒** | 频移（方向/速度） | 血流方向与速度（彩色叠加） | 快速评估血管通畅度、有无反流/狭窄  |
| **频谱多普勒** | 频移（速度定量）  | 血流速度波形（量化分析）   | 精确测量压力差、流量、阻力指数     |
| **弹性成像**   | 应变/剪切波速度   | 组织硬度（彩色编码）       | 辅助鉴别肿瘤良恶性，评估肝纤维化   |
| **超声造影**   | 微泡谐波信号      | 微循环灌注（实时显影）     | 判断肿瘤血供、鉴别坏死与存活组织   |

在临床实际操作中，医生通常是在**B模式**下找到病变位置，然后打开**彩色多普勒**看血流是否丰富，再用**频谱多普勒**测量流速，如果是心脏检查，还会切换**M模式**测量心功能。这些模式相互补充，共同完成诊断。

## 2 超声中是如何通过延时线对波束进行聚焦的

超声波的聚焦，无论是**发射聚焦**还是**接收聚焦**，其核心机制都是利用**延时线**来控制各阵元发射或接收信号的**相位差**。

你可以将探头上的每个阵元想象成独立控制的“小石子”。如果同时丢进水里，产生的是平面波；但如果控制丢石子的时间，让边缘的石子先丢，中间的后丢，产生的波纹就会在预定位置汇聚。

在超声设备中，**延时线**就是用来精确控制这些时间差的关键电路模块。具体实现过程分为发射和接收两个环节：

### 1. 发射聚焦：让声波在目标点“相遇”

发射聚焦的目的是让多个阵元发出的超声波在同一时刻到达目标点，从而在该点产生最强的叠加能量。

- **核心逻辑**：**边缘先发，中间后发**。
- **延时控制过程**：
  假设需要将声束聚焦在深度为 F*F* 的点 P*P*。
  1. **计算距离差**：位于探头边缘的阵元距离 P*P* 点较远，位于中央的阵元距离 P*P* 点较近。
  2. **分配延时**：为了让所有声波同时到达 P*P* 点，距离较远的边缘阵元需要**先**发射，距离较近的中央阵元需要**后**发射。
  3. **执行**：设备中的**发射延时线**为每个通道（阵元）设置一个独立的延迟时间（ΔtΔ*t*）。当触发信号到来时，边缘通道的延时最短（立即执行），中央通道的延时最长。
  4. **结果**：各阵元发出的子波波阵面在传播过程中逐渐由凹面变为平面，最终在 P*P* 点汇聚，形成能量最强、横向分辨率最高的焦点。

### 2. 接收聚焦：动态的“听诊器”

接收聚焦比发射聚焦更为复杂且关键。因为发射时只有一个焦点，但接收时，设备可以对每个深度的回波都进行动态聚焦。

- **核心逻辑**：**近处后听，远处先听**。
- **延时控制过程**：
  当来自某点 P*P* 的回波信号返回探头时：
  1. **到达时间不同**：距离 P*P* 点最近的中央阵元最先接收到回波，边缘阵元最后接收到回波。
  2. **补偿延时**：如果直接将信号叠加，信号会因时间错位而模糊。此时，**接收延时线**发挥作用：给最先到达的中央阵元信号施加**最长的延时**，给最后到达的边缘阵元信号施加**最短的延时**（甚至不加）。
  3. **对齐叠加**：经过延时补偿后，所有通道的信号在时间轴上被“拉齐”，再进行叠加合成。这相当于动态地将接收波束“聚焦”在了那个特定的深度。

### 3. 延时线的技术实现

在现代全数字超声诊断仪中，延时线并非物理上的“线”，而是由**数字电路**和**内存**构成的逻辑单元。

- **采样与存储**：接收到的模拟回声信号经模数转换器转换为数字信号，存入**先进先出（FIFO，First In First Out）** 的内存缓冲区。
- **动态读出**：每个通道对应一个内存区。为了实现聚焦，不同通道的“读指针”会被设置在不同的起始地址。
  - 对于最先到达的信号（中央阵元），系统等一会儿再去读它（延时大）。
  - 对于最后到达的信号（边缘阵元），系统立刻去读它（延时小）。
- **精度**：为了保证图像质量（特别是近场），延时精度通常要求达到**纳秒级**（10−910−9 秒）。

### 4. 发射聚焦 vs. 接收聚焦的差异

| 维度         | 发射聚焦                                                     | 接收聚焦                                                     |
| :----------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| **焦点数量** | **单焦点**（一次发射只能聚焦在一个深度）。如果要多个深度，需多次发射，会降低帧频。 | **动态多焦点**（一次接收过程中，聚焦点随深度实时移动）。     |
| **实现难度** | 相对简单，只需控制发射时序。                                 | 复杂，需要强大的并行计算能力，支持数字延时和动态变迹。       |
| **图像质量** | 决定发射声场的能量集中度。                                   | 决定图像整体的横向分辨率（现代高端超声图像清晰，主要归功于接收聚焦）。 |

### 5. 临床意义：为什么聚焦很重要

如果不使用延时线进行聚焦，超声束会自然扩散，导致：

- **横向分辨率差**：无法区分并排的两个点。
- **对比度降低**：旁瓣干扰严重，图像出现雾状伪像。

通过**延时线控制**，超声设备能够将声束压缩成很细的“笔形波束”。在临床应用中，医生通常会根据病变深度调整**焦点位置**（如通过触摸屏设定焦点深度），其实就是在设置发射聚焦的延时参数，确保目标区域图像最清晰。

**总结**：延时线通过精确控制**边缘阵元先发后收、中央阵元后发先收**的时间补偿机制，利用惠更斯原理，将分散的阵元能量在目标点进行相干叠加，实现了从“手电筒”式散射波束到“激光”式聚焦波束的转变。

## 3 为什么几何分辨率高的时候灰度分辨率会差

简单来说，这种“此消彼长”的现象主要源于**声学物理的限制**、**波束形成的特性**以及**信号处理中的数学矛盾**。我们可以从以下几个维度来拆解：

### 1. 声束聚焦：能量集中与旁瓣干扰的矛盾

高几何分辨率（尤其是横向分辨率）依赖于**极窄的主瓣**和**极低的旁瓣**。

- **为了追求高几何分辨率**：工程师会使用**大孔径**、**强聚焦**的发射方式。这确实能把主瓣压得很窄（分辨细线能力强），但往往会导致**旁瓣电平升高**。
- **旁瓣的代价**：高旁瓣意味着探头不仅从正前方（主瓣）接收信号，还从侧方（旁瓣）接收了不该来的回声。这些旁瓣信号叠加在主瓣信号上，形成“噪声基底”，**掩盖了微弱的回波差异**。
- **结果**：虽然你能看清一根细线的边界（几何分辨率高），但在实质器官（如肝、甲状腺）中，不同组织间微小的灰度差异（如 5 dB 的差别）被旁瓣噪声淹没了，导致**灰度分辨率（对比度）下降**，图像看上去“发花”或“颗粒感重”。

### 2. 信号带宽：脉冲长度与信噪比的博弈

- **轴向分辨率**（沿声束方向的精细度）由**脉冲长度**决定。脉冲越短，轴向分辨率越高。
- **为了追求高轴向分辨率**：发射的超声脉冲必须非常短（宽频带，1-2个周期）。
- **短脉冲的代价**：
  - **总能量低**：脉冲越短，注入人体的总能量越少，回波信号就弱。
  - **信噪比下降**：信号弱意味着回波信号与系统电噪声的比值降低。
  - **灰度分辨率受损**：信噪比是灰度分辨率的基础。当信号微弱时，设备难以区分是“组织真实的细微差异”还是“随机噪声”。为了把图像显示出来，系统被迫压缩灰度动态范围，导致层次感变差。

### 3. 数字信号处理：动态范围与细节提取

在设备后端，存在一个数学上的矛盾：**动态范围压缩**。

- **高几何分辨率的代价**：为了得到极细的聚焦点和清晰的边界，系统往往采用**高频率探头**（频率高，波长短，分辨率高）。
- **高频的物理特性**：高频声波在人体内**衰减极快**。这导致深部组织的回波信号非常微弱。
- **处理的矛盾**：
  - 为了在屏幕上同时显示“高分辨率的浅表组织”和“微弱信号的深部组织”，工程师必须对信号进行**对数压缩**（将数百倍的信号强度差压缩成几十级的灰度显示）。
  - 如果过度强调几何分辨率（高频、强聚焦），深部信号的动态范围会被压缩得更严重，原本能显示 256 级灰度的数据，可能实际只能有效显示 64 级。**灰度分辨率因此恶化**。

### 4. 时间（帧频）与空间（线密度）的权衡

这虽然更多关乎时间分辨率，但也间接影响灰度分辨率。

- **追求极致几何分辨率**：需要增加**线密度**（单位角度内扫描线数量多）和**多焦点发射**（每个深度都聚焦）。
- **代价**：这会大幅降低**帧频**（画面更新速度）。
- **间接影响**：在检查**运动器官（如心脏）** 时，如果帧频过低，为了维持图像连贯性，系统必须牺牲**信号平均次数**（即每根线只能发射 1-2 次）。没有足够的信号平均，随机噪声无法被有效滤除，**灰度分辨率的细腻度**（信噪比）自然下降。

### 5. 物理极限：声束的固有属性

从声学原理上看，两者本身就是矛盾的：

- **几何分辨率**要求声束**窄、短、聚焦好**。
- **灰度分辨率（对比度）** 要求声束**纯净（无旁瓣）、穿透力强（能量高）**。

窄聚焦（强聚焦）虽然让声束变细，但使得**焦区外的声场迅速发散**。在焦区之外，声束变宽、能量分散，不仅几何分辨率下降，由于能量不足，该区域的灰度分辨率（对比度）也会显著劣化。

### 临床中的平衡

在高端超声设备中，工程师不会单纯追求某一项指标的极限，而是根据临床场景做权衡：

- **浅表器官（甲状腺、乳腺）**：**几何分辨率优先**。因为需要看微钙化灶（点状强回声）和边界是否光滑。医生能接受一定程度的噪声，但必须看清细小的解剖结构。
- **腹部（肝、肾）**：**灰度分辨率优先**。因为需要鉴别肝脏的弥漫性病变（脂肪肝、肝硬化），这依赖于对实质回声的“细腻程度”和“均匀度”的判断，而非单纯看清一根细线。
- **心脏**：**时间分辨率优先**。在保证足够帧频的前提下，再平衡几何与灰度。

**总结**：
几何分辨率和灰度分辨率的“此消彼长”，本质上是**能量守恒**与**物理聚焦**在工程实现上的矛盾。高几何分辨率要求将能量**极度压缩在极小的空间内**，这牺牲了**总能量（穿透力）** 和**旁瓣抑制能力**，最终导致系统无法精确分辨微弱的回波差异，从而表现为灰度分辨率的下降。

## 4 dicom是什么

DICOM 是 **医学数字成像与通信标准**的缩写。

简单来说，它是一套全球通用的规则，规定了医学影像如何**存储**、**传输**和**显示**，确保不同品牌、不同设备的影像数据能够互联互通。

### 1. DICOM 的核心组成

DICOM 不仅仅是一种“文件格式”（如 .dcm），它包含了两个关键部分：

| 组成部分         | 含义                                 | 示例                                                         |
| :--------------- | :----------------------------------- | :----------------------------------------------------------- |
| **文件格式**     | 将图像数据与患者信息整合在一个文件中 | 一张超声图像文件中，既包含像素数据（图像），也包含患者姓名、检查日期、设备参数、医院信息等文本数据 |
| **网络通信协议** | 定义了设备之间如何传输数据           | 超声机将图像“发送”到 PACS 系统（影像归档与通信系统），或医生工作站调阅图像，都遵循 DICOM 协议 |

------

### 2. DICOM 文件包含的信息

一张 DICOM 图像远不止是一张“照片”。它包含三个层面的信息：

- **患者信息**：姓名、ID、性别、年龄、出生日期等。
- **检查信息**：检查日期、设备型号、操作医生、检查部位等。
- **图像/技术参数**：这是临床和科研中非常重要的部分。对于超声图像，DICOM 头文件中会记录：
  - **探头频率**、**探头类型**
  - **机械指数**、**热指数**
  - **增益**、**动态范围**、**焦点位置**
  - **多普勒角度**、**流速标尺**
  - **测量值**（如心腔大小、血流速度等）

这意味着，当你打开一张超声 DICOM 图像时，医生可以回溯检查时的所有设置，确保诊断依据可追溯。

------

### 3. DICOM 在超声中的特殊之处

与 CT、MRI 等静态图像不同，超声具有**动态实时**的特点。DICOM 标准为此提供了两种支持：

- **静态图像**：单帧 DICOM 文件，类似于一张照片。
- **动态图像**：**DICOM 多帧文件**。超声的 CINE 回放片段可以保存为一个包含连续数百帧的 DICOM 文件，医生可以逐帧回顾整个心动周期或血流变化。

------

### 4. 为什么 DICOM 很重要

在没有 DICOM 标准的时代，每家设备厂商使用自己的专有格式，医院内部无法跨设备调阅图像，患者也只能带着胶片就诊。

有了 DICOM 标准后：

- **互联互通**：GE、飞利浦、西门子等不同品牌的超声机、CT、核磁，都可以将图像统一发送到 **PACS 系统**。
- **无纸化/无胶片化**：医生在诊断工作站即可调阅全部影像，无需打印胶片，且可以调节窗宽窗位、放大、测量等。
- **远程会诊**：基层医院的超声图像可以通过 DICOM 协议传输至上级医院专家端，专家看到的图像质量与原始设备完全一致（包含完整参数）。
- **科研与教学**：研究者可以批量分析 DICOM 文件中的定量数据（如测量值、设备参数），而不只是“看图说话”。

------

### 5. 常见误区

| 误区                     | 说明                                                         |
| :----------------------- | :----------------------------------------------------------- |
| **DICOM 就是 .jpg**      | 不对。.jpg 是压缩图片，丢失了患者信息、设备参数和测量数据。DICOM 包含完整的元数据，且可以无损传输 |
| **超声存图直接截图就行** | 截图（如手机拍屏幕）不是 DICOM，无法用于正规诊断报告或远程会诊。标准流程应保存 DICOM 格式的原始图像或片段 |
| **只有放射科用 DICOM**   | 不对。超声、内镜、病理等所有医学影像科室都使用 DICOM 标准    |

------

### 6. 一个典型的工作流程

1. **图像采集**：超声医生在设备上完成检查，冻结图像或保存 CINE 片段。
2. **存储**：超声机将图像保存为 DICOM 格式（.dcm），可能存储在本地硬盘或直接发送至 PACS 服务器。
3. **传输**：通过医院网络（DICOM 协议），图像被发送到医生的诊断工作站、临床科室或远程会诊平台。
4. **报告**：医生在 PACS 上打开 DICOM 图像，进行测量、标注，并生成图文报告。
5. **归档**：所有 DICOM 图像长期存储在医院数据中心，符合医疗法规要求。

------

**总结**：DICOM 是医学影像领域的“通用语言”。对于超声而言，它不仅是保存图像的标准，更是确保**诊断信息完整、跨平台可追溯、临床工作流顺畅**的基础。无论是日常报告、远程会诊还是科研分析，DICOM 都是不可或缺的一环。
